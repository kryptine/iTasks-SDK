This is the repository of the iTask System (iTasks).

The repository is organized in the following folders:

- Language : All Clean libraries required for programming workflow specifications.
           : This effectively defines the iTasks workflow language.

- Server   : The Clean framework that executes the programming specifications

- Client   : The Javascript default client application
           : IMPORTANT: If you obtained iTasks directly from the subversion repository
           :            you need no run Client/build.bat to compile the client application.

- Daemon   : The daemon application that automates background tasks.

- Examples : Example workflow suites.

- Tests    : Small test programs.

- Tools    : Various build and code generation tools.
